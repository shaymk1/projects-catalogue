In 2025 i ran blog

This is a blog to document my running journey. It is simple typical blog where it has a home page, a blog page, a way to add, delete or update blogs. I have used third party emailing and marketing app called Brevo to manage my subscribers.

Features:
  • User Authentication & Authorization
  • CRUD operations for blog posts
  • Email newsletter subscription
  • Responsive design
Tech Stack:
  • Django & Django REST Framework
  • PostgreSQL Database
  • Bootstrap 5
  • Brevo Email Marketing API
What I Learned:
  • Integration with third-party APIs
  • User authentication best practices
  • Database relationship modeling
  • Email template design and automation
